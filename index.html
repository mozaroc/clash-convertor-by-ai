Конвертер ссылок → YAML для Mihomo (с шаблонами)

Открыть папку templates/
Шаблоны лежат в папке templates/ рядом с index.html. Список читается из templates/index.json. При открытии по file:// появится кнопка выбора локальных файлов.
Скрипт конвертирует ссылки в YAML-блок proxies: и вставит/заменит его в выбранном шаблоне.
Будет создана секция proxy-providers: с типом http и ссылками на подписки. Mihomo сам будет ходить по этим URL и обновлять список прокси.
Нет WARP и переключателей — только dropdown шаблонов, proxies из ссылок и/или proxy-providers из подписок.
